Здоавствуйте, уважаемые жюри!
Представляю Вашему вниманию игру в космической тематике "Space Shooter"!
На момент сдачи решения на 10 баллов реализован следущий MVP:
--Главная игровая задача - выживание: улонение/уничтожение втречающихся на пути астероидах.
--В игре пристутствуют три сцены: Главное меню, главная игровая сцена и сцена проигрыша.
--Управление кораблем возможно только по оси x (с ограничениями) с помощью двух кнопок.
--У корабля есть возможность уничтожать астеориды лазерным оружием (однако
будте осторожны, между выстрелами требуется время на перезарядку).
--В случае проигрыша игрока перекидывает на сцену с уведомлением и возможностью
выйти в главное меню
--Для каждой механики есть звуки!!!
