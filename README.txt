Здоавствуйте, уважаемые жюри!
Представляю Вашему вниманию игру в космической тематике "Space Shooter"!


На момент сдачи решения на 10 баллов реализован следущий MVP:
--(MVP)Главная игровая задача - выживание: улонение/уничтожение втречающихся на пути астероидах.
--(MVP)В игре пристутствуют три сцены: Главное меню, главная игровая сцена и сцена проигрыша.
--(MVP)Управление кораблем возможно только по оси x (с ограничениями) с помощью двух кнопок.
--(MVP)У корабля есть возможность уничтожать астеориды лазерным оружием (однако
будте осторожны, между выстрелами требуется время на перезарядку).
--(MVP)В случае проигрыша игрока перекидывает на сцену с уведомлением и возможностью
выйти в главное меню
--(MVP)Для каждой механики есть звуки!!!
Таким образом был создан MVP игры про космический корабль
