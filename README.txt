Представляю Вашему вниманию игру в космической тематике "Space Shooter"!
(Для кадой версии игры есть своя .apk ссылка, однако приложение имеет одно название, поэтому

при сохранении)
P.S. некоторые формулировки в описании выполненного задания на конкретное количество баллов могли менятся
(только в плане дополнительного описания, а не изменения решения), так что рекомендуется читать финальную
версию README.


На момент сдачи решения на 10 баллов реализован следущий MVP:
--(MVP)Главная игровая задача - выживание: улонение/уничтожение втречающихся на пути астероидах.
--(MVP)В игре пристутствуют три сцены: Главное меню, главная игровая сцена и сцена проигрыша.
--(MVP)Управление кораблем возможно только по оси x (с ограничениями) с помощью двух кнопок.
--(MVP)У корабля есть возможность уничтожать астеориды лазерным оружием (однако
будте осторожны, между выстрелами требуется время на перезарядку) Для выстрела нажмите на свободную часть
экрана.
--(MVP)В случае проигрыша игрока перекидывает на сцену с уведомлением и возможностью
выйти в главное меню
--(MVP)Для каждой механики есть звуки!!!
Таким образом, был создан MVP игры про космический корабль

На момент сдачи решения на 20 баллов реализован следущий MVP:
--(ОБНОВЛЕНИЕ СИСТЕМЫ УПРАВЛЕНИЯ)Добавлен гипер прыжок! Теперь с интервалом в 3 секунды корабль
может мгновенно перемещаться по оси x на некоторое не очень большое расстояние для лучшего уклонения.
Чтобы воспользоваться данной функицей корабля нужно сделать двойной щелчок по правой или левой кнокпе
управления. (Прыжок осуществляется с характерным звуком)
--(ОБНОВЛЕНИЕ СИСТЕМЫ УПРАВЛЕНИЯ)Добавлено перемещение по вертикали! Теперь космиечский корабль может
перемещаться еще и по оси y (с ограничениями) для лучшего уклонения (добавлены нужные кнопки управления).
--(ВЗАИМОДЕЙСТВИЕ С ОКРУЖАЮЩИМ МИРОМ)Добавлены обломки кораблей! Теперь бороздя космос игрок может
собирать останки других лететельных аппаратов! В правом верхнем углу располагается игровой счет
(Обломки считаются постоянной валютой, которая не обнуляется при смерти игрока)
(в последющем за счет обломков можно будет прокачивать свой корабль)
Таким образом, была обогощена система управления игроком и добавлено взаимодействие с окружающим 
миром

На момент сдачи решения на 30 баллов реализован следущий MVP:
--(ПРОКАЧКА ИГРОКА) Теперь по мере исследования космоса игрок улучшает свои навыки!
Механика реализована следующим образом: 
-каждые 5 секунд у игрока повышается скорость на 0.1 у.е.
-каждые 15 секунд у игрока уменьшается время на перезарядку на 0.01 у.е. 
(P.S. в последующих версиях значения бонусов и интервалы могут калиброваться)
--(ПРОКАЧКА ИГРОКА) Теперь игроку доступен магазин с возможностью купить улучшения за найденные обломки!
(На момент MVP есть только один слот, да и тот с глюком - суть системы в том чтобы показать какого
вида улучшения можно купить. Купленная скорость x2 означает что в игре у играка будет сразу двойная скорость)
Таким образом, была обогощена система управления игроком и добавлено взаимодействие с окружающим 
миром
